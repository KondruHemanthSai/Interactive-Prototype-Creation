# 📱 TaskFlow – Interactive Prototype

## 📌 Overview
TaskFlow is a modern mobile task management application prototype designed to help users organize, prioritize, and track their daily tasks efficiently. This interactive prototype demonstrates the complete user flow from launching the application to managing and completing tasks.

---

## 🎯 Objective
The objective of this prototype is to showcase a clean and intuitive user interface while simulating real application interactions. It focuses on improving user experience through simple navigation, clear layouts, and interactive screens.

---

## ✨ Features

- Splash Screen
- User Login Screen
- Home Dashboard
- Add New Task
- View Completed Tasks
- Task Priority Selection
- Reminder Toggle
- Interactive Navigation
- Modern Material Design UI

---

## 📱 Prototype Screens

### 🚀 Splash Screen
- Displays the TaskFlow logo and branding.
- Welcomes users before entering the application.
- Continue button navigates to the Login screen.

---

### 🔐 Login Screen
- Email and Password input fields.
- Login button.
- Forgot Password option.
- Create Account link.

---

### 🏠 Home Screen
- Personalized greeting.
- Task summary cards.
- Displays pending and completed task counts.
- Lists today's tasks with priority labels.
- Floating Action Button to add new tasks.
- Bottom navigation bar.

---

### ➕ Add Task Screen
- Enter task name.
- Select due date.
- Choose task time.
- Select task priority (Low, Medium, High).
- Optional notes section.
- Reminder toggle.
- Save and Cancel actions.

---

### ✅ Completed Tasks Screen
- Displays completed tasks.
- Shows task category.
- Completion status.
- Easy navigation back to previous screens.

---

## 🎨 Design Highlights

- Modern minimalist interface
- Material Design inspired components
- Consistent blue color palette
- Rounded cards and buttons
- Responsive mobile layout
- Clear typography
- Interactive transitions between screens
